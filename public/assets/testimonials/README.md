# Testimonials Images

## Localização das Imagens

Coloque as imagens dos depoimentos nesta pasta (`public/assets/testimonials/`).

## Arquivos necessários:

1. **marina-souza.jpg** - Imagem da Marina de Souza (Empreendedora)
2. **joao-pedro.jpg** - Imagem do João Pedro (Freelancer)
3. **ana-costa.jpg** - Imagem da Ana Costa (Gerente Financeira)

## Especificações recomendadas:

- **Formato**: JPG ou PNG
- **Tamanho**: 200x200px (mínimo) a 400x400px (recomendado)
- **Proporção**: Quadrada (1:1)
- **Qualidade**: Boa resolução para exibição em tela retina
- **Fundo**: Preferencialmente transparente ou fundo neutro

## Nota:

Se as imagens não forem encontradas, o sistema usará automaticamente o placeholder padrão (`/placeholder.svg`).
