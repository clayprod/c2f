# Apple Touch Icon

Este arquivo deve ser uma imagem PNG de 180x180 pixels.

## Como criar:

1. Use o mesmo favicon ou logo da aplicação
2. Redimensione para exatamente 180x180 pixels
3. Salve como `apple-touch-icon.png` nesta pasta (`public/`)

## Formato:
- **Tamanho**: 180x180 pixels
- **Formato**: PNG
- **Localização**: `public/apple-touch-icon.png`

O Next.js já está configurado para servir este arquivo automaticamente quando ele existir.
